Projects

Grade Distribution - This platform enables teachers to distribute student grades using a single xlxs spreadsheet when logged in as an admin account. Using student's identification number, parents can log in as guests, they are allowed to review the exam results of their own child only. This project uses django rest framework, redux and webpack.

Annotation Platform - Annotation/Training data management site, providing three main features including 1) conditional data retrieval, 2) data auto archieve and 3) Real time review/fix of data annotation. Used Docker, Nginx, Gunicorn, Django and PostgresDB.

Skylark - Predicting Proto-planetary disks using deep learning. Msc project for frame extrapolation on astronomical simulation data. Used PyTorch and was trained on Compute Canada. Currently focusing on 1) make the network to inherites sequential information such as momantum. 2) dataset expansion.

Cloud Deep Learning - A system similar to Google Vizier which performs automated parallel hyperparameter searching (Grid, Random and Bayesian Optimization) across clusters.

Full-Resolution-ResNet - Tensorflow implementation of the Full-Resolution-ResNet on cityspace data.

Sponza - A real-time renderer that stimulates a sponza structure with advanced lighting and skybox implemented in c++ and openGL.
